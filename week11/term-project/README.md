## Week 11 Term Project Update

Expanded my Week 10 Hawaii hospitality project by adding an embedded Review schema to the Property model using Mongoose. I created 3 Express routes for listing all properties, viewing a single property by ID, and adding reviews to a property. 

I also built a basic EJS page at `/properties` that displays all seeded properties from MongoDB. This week helped me understand how Mongoose schemas, Express routes, and server-side rendering work together as a basic pipeline from the UI to the database.
