# Week 10 Deliverable - Term Project 3
## Property Schema & Seed Data

### Project Overview
For this project, I selected **Oahu** as my Hawaiian island and created a fictional hotel concept called **Pacific Horizon Hotel**. It is a **hotel** located in **Waikiki, Oahu**. The target visitor segment is **business travelers, couples, and first-time visitors**.

This Week 10 deliverable focuses on:
- defining the property concept
- creating a Mongoose schema for the Property model
- inserting sample property records into MongoDB
- verifying the MongoDB connection using MongoDB Atlas



## Property Information
- **Property Name:** Pacific Horizon Hotel
- **Island:** Oahu
- **Type:** Hotel
- **Location:** Waikiki, Oahu
- **Target Segment:** Business travelers, couples, and first-time visitors

### Property Description
Pacific Horizon Hotel is a modern hotel in Waikiki designed for guests who want convenient access to beaches, shopping, dining, and Honolulu attractions. It is intended for business travelers, couples, and first-time visitors seeking a comfortable and well-located stay on Oahu.
