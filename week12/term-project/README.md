# Term Project – Week 12 React Marketing Page

## Project Overview
This term project continues the development of my hospitality property application by adding a React-based marketing page UI. My fictional property is Pacific Horizon Hotel, located in Waikiki, Oahu. The property is designed for business travelers, couples, and first-time visitors who want a modern hotel with convenient access to beaches, dining, shopping, and local attractions.


## Connection to Previous Weeks
In previous weeks, I developed the foundation for the term project by defining the property concept, planning the schema, and organizing the project requirements. Week 12 builds on that work by creating the React user interface and turning the project plan from the PRD into a working marketing page layout.


## Features Implemented
- React-based marketing page
- Hero section with hotel name and tagline
- About section describing the property
- Amenities section listing hotel features
- Call-to-Action section with booking prompt
- Screenshot of the rendered page saved in the `docs/` folder


## How to Run the Project
1. Open the terminal in `week12/term-project`
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
4. Open the local browser link shown in the terminal


## Reflection
For this week, I built the initial React marketing page for my term project and used the PRD as the guide for organizing the layout and content. I decided to structure the page into clear sections for the hero, about, amenities, and call-to-action areas because that made the interface easier to read and closer to a real hotel marketing page. I also kept the layout simple so it would be easy to review and run. This week helped me connect the earlier planning and project foundation to a working front-end interface.
