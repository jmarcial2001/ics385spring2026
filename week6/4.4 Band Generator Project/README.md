# UHMC Course Generator
### This is a small Express and EJS web app that randomly generates a UHMC course when the button is clicked.
The server selects a random course ID and course name, then renders it on the page using EJS.
- Node.js
- Express
- EJS
### How to Run (CodeSandBox)
1. Navigate to the project folder
2. Run:
npm install
npm run start
The app runs on port 3000 (or the environment port).
